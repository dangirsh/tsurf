  <name>Task 1: Create claw-swap NixOS module with hardened containers</name>
  <files>
    /data/projects/claw-swap/nix/module.nix

  <action>
    Create `nix/module.nix` following the parts module pattern exactly (curried module, no sops-nix import, no system-level config). This module declares the entire claw-swap stack.

    **Module structure** (curried pattern -- outer args from flake, inner args from NixOS):
    ```nix
    { self, sops-nix, ... }:
    { config, lib, pkgs, ... }:
    ```

    **Sections to implement (in order):**

    **1. Secrets (5 entries):**
    - `claw-swap-db-password` -- restartUnits: db + app
    - `claw-swap-r2-account-id` -- restartUnits: app
    - `claw-swap-r2-access-key-id` -- restartUnits: app
    - `claw-swap-r2-secret-access-key` -- restartUnits: app
    - `claw-swap-world-id-app-id` -- restartUnits: app
    - All use `sopsFile = self + "/secrets/claw-swap.yaml"` (per-secret override, NOT defaultSopsFile)

    **2. Secret Templates (2 env files):**
    - `claw-swap-db-env`: `POSTGRES_PASSWORD=${config.sops.placeholder."claw-swap-db-password"}`
    - `claw-swap-app-env`: DATABASE_URL (constructed with db password placeholder), R2 creds, World ID app ID

    **3. Pulled Docker Images:**
    - PostgreSQL: `dockerTools.pullImage { imageName = "postgres"; finalImageTag = "16-alpine"; ... }`
    - Caddy: `dockerTools.pullImage { imageName = "caddy"; finalImageTag = "2-alpine"; ... }`
    - Use `nix-prefetch-docker --image-name postgres --image-tag 16-alpine --quiet` to get imageDigest + sha256
    - Use `nix-prefetch-docker --image-name caddy --image-tag 2-alpine --quiet` to get imageDigest + sha256

    **4. Caddyfile:**
    ```nix
    caddyfile = pkgs.writeText "Caddyfile" ''
      claw-swap.com {
        reverse_proxy claw-swap-app:3000
      }
    '';
    ```
    Caddy TLS/domain config ownership is in the claw-swap repo per Claude's discretion (domain is tightly coupled to the app).

    **5. Docker Network:**
    - `docker-network-claw-swap-net` systemd oneshot (same pattern as parts)
    - Bridge driver, subnet 172.22.0.0/24

    **6. Containers (3 total) -- ALL with security hardening:**

    **claw-swap-db (PostgreSQL 16):**
    - image: "postgres:16-alpine", imageFile: postgresImage
    - environment: POSTGRES_DB="claw_swap", POSTGRES_USER="claw"
    - environmentFiles: sops template for db password
    - volumes: `/var/lib/claw-swap/pgdata:/var/lib/postgresql/data`
    - extraOptions (hardening):
      - `--read-only`
      - `--tmpfs=/tmp:rw,noexec,nosuid`
      - `--tmpfs=/run/postgresql:rw,noexec,nosuid` (PostgreSQL socket/PID file)
      - `--cap-drop=ALL`
      - `--cap-add=CHOWN`, `--cap-add=SETUID`, `--cap-add=SETGID`, `--cap-add=FOWNER`, `--cap-add=DAC_OVERRIDE` (PostgreSQL initdb needs these)
      - `--security-opt=no-new-privileges`
      - `--memory=512m`, `--cpus=1.0`
      - `--shm-size=128m` (PostgreSQL shared memory)
      - `--restart=on-failure:5`
      - `--network=claw-swap-net`

    **claw-swap-app (Hono + Node.js 22):**
    - image: "claw-swap:latest", imageFile: via pkgs.callPackage (self + "/nix/claw-swap-app.nix") { src = self; }
    - environment: NODE_ENV, PORT, APP_BASE_URL, BASE_URL, WORLD_ID_ACTION, R2_BUCKET_NAME (non-secret values)
    - environmentFiles: sops template for app secrets (DATABASE_URL, R2, World ID)
    - dependsOn: ["claw-swap-db"]
    - extraOptions (hardening):
      - `--read-only`
      - `--tmpfs=/tmp:rw,noexec,nosuid`
      - `--cap-drop=ALL`
      - `--security-opt=no-new-privileges`
      - `--memory=512m`, `--cpus=1.0`
      - `--restart=on-failure:5`
      - `--network=claw-swap-net`

    **claw-swap-caddy (Caddy 2):**
    - image: "caddy:2-alpine", imageFile: caddyImage
    - volumes: Caddyfile (read-only), caddy-data, caddy-config
    - ports: ["80:80", "443:443"]
    - dependsOn: ["claw-swap-app"]
    - extraOptions (hardening):
      - `--read-only`
      - `--tmpfs=/tmp:rw,noexec,nosuid`
      - `--cap-drop=ALL`
      - `--cap-add=NET_BIND_SERVICE` (ports 80/443)
      - `--security-opt=no-new-privileges`
      - `--memory=256m`, `--cpus=0.5`
      - `--restart=on-failure:5`
      - `--network=claw-swap-net`

    **7. Systemd Ordering:**
    - All 3 container services: `after` + `requires` on `docker-network-claw-swap-net.service`

    **8. Host Directories (tmpfiles.rules):**
    - `/var/lib/claw-swap` (root dir)
    - `/var/lib/claw-swap/pgdata`
    - `/var/lib/claw-swap/caddy-data`
    - `/var/lib/claw-swap/caddy-config`

    **Module MUST NOT:**
    - Import sops-nix module
    - Set sops.defaultSopsFile
    - Enable Docker
    - Set firewall rules
    - Set virtualisation.oci-containers.backend (this is already set in parts module; setting it in both is fine as long as both say "docker" -- but better to let agent-neurosys own it if there's a conflict)

    **Resource limits summary (per user decision -- generous headroom on 47GB VPS):**
    - App: 512MB memory, 1.0 CPU
    - DB: 512MB memory, 1.0 CPU, 128MB shared memory
    - Caddy: 256MB memory, 0.5 CPU

    After writing, `git add nix/module.nix` and run `nix-prefetch-docker` for both images to get the real hashes. Update the placeholder hashes. Then `cd /data/projects/claw-swap && nix flake check` to validate.

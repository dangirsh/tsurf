**Known open questions from research (expect to resolve here):**
- bwrap user namespace behavior on NixOS (--unshare-user + --disable-userns)
- Rootless Podman inside bwrap user namespace (--disable-userns may conflict)
- NixOS /etc/static symlink resolution through bwrap --ro-bind
- zmx socket location inside sandbox (may need /run/user bind-mount)

**If --disable-userns breaks Podman:**

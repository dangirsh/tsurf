
If `neurosys-prod` doesn't resolve via MagicDNS, the hostname in the Tailscale admin panel may differ. Check `tailscale status` for the actual name.

**Step 3: Full service verification**

Run comprehensive checks via Tailscale:
```bash
ssh root@neurosys-prod bash -s <<'VERIFY'
echo "=== System ==="
systemctl is-system-running
echo ""

echo "=== Failed units ==="
systemctl --failed --no-pager
echo ""

echo "=== Secrets ==="
ls /run/secrets/
echo ""

echo "=== Docker ==="
docker ps --format 'table {{.Names}}\t{{.Status}}'
echo ""

echo "=== Firewall ==="
nft list ruleset | head -40
echo ""

echo "=== Tailscale ==="
tailscale status
echo ""

echo "=== Fail2ban ==="
systemctl is-active fail2ban
echo ""

echo "=== Prometheus ==="
curl -s http://localhost:9090/-/healthy
echo ""

echo "=== SSH port NOT on public interface ==="
ss -tlnp | grep :22
echo ""
VERIFY
```

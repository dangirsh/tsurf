# Quickstart Overlay Template

Generated by `tsurf init` for a fast local-only evaluation flow.

- Lives under `.tsurf/overlay/` by default and is ignored by git.
- Targets an existing NixOS host reachable over SSH as `root`.
- Keeps the production/private-overlay path available for later, but hides it
  while you are just trying tsurf out.

The generated overlay imports a lighter module set than the full production
template so you can deploy onto a vanilla NixOS server without first adopting
the full disk/boot layout from this repo.
